## Installation

::code-group
  ```bash [Yarn]
  yarn add @vlalg-nimbus/nb-pagination
  ```
  ```bash [NPM]
  npm install @vlalg-nimbus/nb-pagination
  ```
::

## Usage

::code-group
  ```js [Vue 3]
  import { createApp } from 'vue'
  import App from './App.vue'

  import NbPaginationComponents from '@vlalg-nimbus/nb-pagination'
  import "@vlalg-nimbus/nb-pagination/dist/style.css";

  const app = createApp(App)
  app.use(NbPaginationComponents)
  app.mount('#app')
  ```
  ```js [Nuxt 3]
  import NbPaginationComponents from '@vlalg-nimbus/nb-pagination'
  import "@vlalg-nimbus/nb-pagination/dist/style.css";
  
  export default defineNuxtPlugin(context => {
    context.vueApp.use(NbPaginationComponents)
  })
  ```
::

To use, simply call the component, in this case it will be `NbPagination` or `nb-pagination`.

## Events

### update:modelValue

Fired when the current page changes (required for v-model binding).

**Returns:**
- `Number` - The new page number

**Example:**
```javascript
{
  // Emitted value: 5 (when user navigates to page 5)
}
```

---

### changed

Fired when the current page changes (alternative to `update:modelValue`).

**Returns:**
- `Number` - The new page number

**Example:**
```javascript
{
  // Emitted value: 5 (when user navigates to page 5)
}
```

---

### page-changed

Fired when the current page changes, providing detailed information about the change.

**Returns:**
```javascript
{
  previousPage: Number,    // Previous page number
  currentPage: Number      // New page number
}
```

**Example:**
```javascript
{
  previousPage: 3,
  currentPage: 5
}
```

---

### go-to-input

Fired when the user types in the "Go to" input field.

**Returns:**
```javascript
{
  value: String,           // Numeric value (cleaned, only digits)
  rawValue: String         // Original value typed by user
}
```

**Example:**
```javascript
{
  value: "12",             // Cleaned numeric value
  rawValue: "12a"          // Original input (non-numeric chars removed)
}
```

---

### invalid-page

Fired when the user attempts to navigate to an invalid page (outside the valid range).

**Returns:**
```javascript
{
  page: Number,            // The invalid page number attempted
  min: Number,             // Minimum valid page (always 1)
  max: Number              // Maximum valid page (totalPages)
}
```

**Example:**
```javascript
{
  page: 15,                // User tried to go to page 15
  min: 1,                  // Minimum valid page
  max: 12                  // Maximum valid page
}
```

---

### first-page

Fired when the user navigates to the first page.

**Returns:**
- `Number` - The current page number (always 1)

**Example:**
```javascript
{
  // Emitted value: 1 (when user navigates to first page)
}
```

---

### last-page

Fired when the user navigates to the last page.

**Returns:**
- `Number` - The current page number (equals totalPages)

**Example:**
```javascript
{
  // Emitted value: 12 (when user navigates to last page, assuming totalPages is 12)
}
```

---

### scrolling

Fired when automatic scrolling is executed after a page change (only when `scrollOnPageChange` is `true`).

**Returns:**
```javascript
{
  target: String,           // Scroll target: 'top', 'component', or 'custom'
  selector: String | null,  // CSS selector used (when target='custom')
  offset: Number,           // Offset in pixels applied
  behavior: String,         // Scroll behavior: 'smooth' or 'auto'
  page: Number,             // Current page number after change
  position: Number,         // Final scroll position in pixels
  elementId: String | undefined  // Element ID (when target='component')
}
```

**Example:**
```javascript
// When scrollTarget="top"
{
  target: 'top',
  selector: null,
  offset: 0,
  behavior: 'smooth',
  page: 5,
  position: 0
}

// When scrollTarget="component"
{
  target: 'component',
  selector: null,
  offset: 0,
  behavior: 'smooth',
  page: 5,
  position: 450,
  elementId: 'pagination-1'
}

// When scrollTarget="custom" with scrollSelector="#header"
{
  target: 'custom',
  selector: '#header',
  offset: 50,
  behavior: 'smooth',
  page: 5,
  position: 200
}
```

**Note:** This event is only emitted when:
- `scrollOnPageChange` is `true`
- The scroll action is successfully executed
- The target element is found (for `component` and `custom` targets)

**Example Usage:**
```vue
<template>
  <NbPagination
    nb-id="pagination-1"
    v-model="currentPage"
    :total-pages="12"
    :scroll-on-page-change="true"
    scroll-target="top"
    scroll-behavior="smooth"
    @scrolling="handleScrolling"
  />
</template>

<script setup>
import { ref } from 'vue'

const currentPage = ref(1)

const handleScrolling = (scrollData) => {
  console.log('Scroll executed:', scrollData)
  // You can track scroll events, show loading indicators, etc.
}
</script>
```

## Slots

### slim-text

Custom slot for displaying custom text when `showSlimText` is `true`.

**Slot Props:**
```javascript
{
  currentPage: Number,     // Current page number
  totalPages: Number       // Total number of pages
}
```

**Example:**
```vue
<template #slim-text="{ infos }">
  <span>Página {{ infos.currentPage }}/{{ infos.totalPages }}</span>
</template>
```

---

### go-to-select

Custom slot for the "Go to" select dropdown when `goToType="select"`.

**Slot Props:**
```javascript
{
  currentPage: Number,           // Current page number
  totalPages: Number,            // Total number of pages
  disabled: Boolean,             // Whether the component is disabled
  goToChangedSelect: Function    // Function to call when selection changes
}
```

**Example:**
```vue
<template #go-to-select="{ infos }">
  <select @change="infos.goToChangedSelect" :disabled="infos.disabled">
    <option
      v-for="i in infos.totalPages"
      :key="i"
      :value="i"
      :selected="i === infos.currentPage"
    >
      {{ i }}
    </option>
  </select>
  <span> de {{ infos.totalPages }}</span>
</template>
```

### With Scroll on Page Change

```vue
<template>
  <!-- Scroll to top of page -->
  <NbPagination
    nb-id="pagination-scroll-top"
    v-model="currentPage"
    :total-pages="12"
    :scroll-on-page-change="true"
    scroll-target="top"
    scroll-behavior="smooth"
    :scroll-offset="100"
  />

  <!-- Scroll to pagination component -->
  <NbPagination
    nb-id="pagination-scroll-component"
    v-model="currentPage"
    :total-pages="12"
    :scroll-on-page-change="true"
    scroll-target="component"
    scroll-behavior="smooth"
  />

  <!-- Scroll to custom element by ID -->
  <NbPagination
    nb-id="pagination-scroll-custom-id"
    v-model="currentPage"
    :total-pages="12"
    :scroll-on-page-change="true"
    scroll-target="custom"
    scroll-selector="#header"
    scroll-behavior="smooth"
    :scroll-offset="50"
  />

  <!-- Scroll to custom element by class -->
  <NbPagination
    nb-id="pagination-scroll-custom-class"
    v-model="currentPage"
    :total-pages="12"
    :scroll-on-page-change="true"
    scroll-target="custom"
    scroll-selector=".content-wrapper"
    scroll-behavior="smooth"
  />

  <!-- Scroll to custom element by CSS selector -->
  <NbPagination
    nb-id="pagination-scroll-custom-selector"
    v-model="currentPage"
    :total-pages="12"
    :scroll-on-page-change="true"
    scroll-target="custom"
    scroll-selector="[data-scroll-to]"
    scroll-behavior="smooth"
  />
</template>

<script setup>
import { ref } from 'vue'

const currentPage = ref(1)
</script>
```

**Scroll Options:**

- **`scrollTarget="top"`**: Scrolls to the top of the page (useful for long content pages)
- **`scrollTarget="component"`**: Scrolls to the pagination component itself
- **`scrollTarget="custom"`**: Scrolls to a custom element specified by `scrollSelector`

**Scroll Selector Examples:**

- ID: `"#header"` or `"header"` (without `#`)
- Class: `".content-wrapper"` or `".my-class"`
- CSS Selector: `"div.container"`, `"[data-scroll-to]"`, `"main > section"`, etc.

**Use Cases:**

- **Long content pages**: Use `scrollTarget="top"` to scroll to top after page change
- **Fixed headers**: Use `scrollOffset` to account for fixed navigation bars
- **Custom scroll points**: Use `scrollTarget="custom"` with `scrollSelector` to scroll to specific sections
