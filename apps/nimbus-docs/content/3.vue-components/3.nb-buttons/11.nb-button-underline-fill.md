# nb-button-underline-fill

This is an underline fill button component for [Vue.js 3+](https://vuejs.org/).

::NbButtonUnderlineFillTest
::

## Installation

::code-group
  ```bash [Yarn]
  yarn add @vlalg-nimbus/nb-buttons
  ```
  ```bash [NPM]
  npm install @vlalg-nimbus/nb-buttons
  ```
::

## Usage

::code-group
  ```js [Vue 3]
  import { createApp } from 'vue'
  import App from './App.vue'

  import NbButtonsComponents from '@vlalg-nimbus/nb-buttons'
  import "@vlalg-nimbus/nb-buttons/dist/style.css";

  const app = createApp(App)
  app.use(NbButtonsComponents)
  app.mount('#app')
  ```
  ```js [Nuxt 3]
  import NbButtonsComponents from '@vlalg-nimbus/nb-buttons'
  import "@vlalg-nimbus/nb-buttons/dist/style.css";
  
  export default defineNuxtPlugin(context => {
    context.vueApp.use(NbButtonsComponents)
  })
  ```
::

To use, simply call the component, in this case it will be `NbButtonUnderlineFill` or `nb-button-underline-fill`.

::code-group
  ```js [Mode 1]
  <template>
    <NbButtonUnderlineFill />
  </template>
  ```
  ```js [Mode 2]
  <template>
    <nb-button-underline-fill />
  </template>
  ```
  ```js [Mode 3]
  <template>
    <nb-button-underline-fill></nb-button-underline-fill>
  </template>
  ```
::

## Props

> Items with an (*) mean they are required

| name    | Value type | Default | Description |
| ------- | :--------- | :------ | :---------------------------- |
| nbId (*)    | String     |        | Sets the id attribute to differentiate from other components |
| display     | String     | 'b'    | Defines the display type. Accepts **ib** and **b**. |
| tabIndex     | Number     | 0      | Defines the tab index. Set this property to make the button focusable by the keyboard. |
| hasTabIndexEnter     | Boolean     | true      | Enables the button to be activated with the Enter key when focused. |
| hasTabIndexSpace     | Boolean     | true      | Enables the button to be activated with the Space key when focused. |
| ariaLabel     | String     | 'Alternate Text Button'      | Defines the aria-label attribute for screen readers. |
| textColor   | String     | 'rgb(0, 105, 255)'  | Defines the text color. Accepts **Color name** and **Hex** |
| textHoverColor   | String     | '#ffffff'  | Defines the text color on hover. Accepts **Color name** and **Hex** |
| buttonColor   | String     | 'rgb(0, 105, 255)'  | Defines the button background color (starts as a thin line at the bottom). Accepts **Color name** and **Hex** |
| buttonHoverColor   | String     | 'rgb(20, 51, 214)'  | Defines the button background color on hover (fills the entire button). Accepts **Color name** and **Hex** |
| textAlign   | String     | 'middle'  | Defines the text alignment. Accepts **left**, **center**, **right**. |
| width    | Number     |  86      | Defines button width. |
| paddingX    | Number     |  1      | Defines button padding-left and padding-right. |
| paddingY    | Number     |  0.2    | Defines button padding-top and padding-button. |
| disabled    | Boolean    |  false  | Defines if the button is disabled  |
| fontFamily   | String     | "'Lato', sans-serif"  | Defines the font-family |
| fontSize   | String     | '1.6em'  | Defines the font-size |
| fontWeight   | Number     | 200  | Defines the font-weight |

## Events

| name    | Return type | Description |
| ------- | :--------- | :---------------------------- |
| clicked | nothing | Fired when the button is clicked, returns nothing. |

## Slot

The component has a slot called `content` where the content that will be manipulated must be passed. It has a default text (`Default Text`) in case the content does not pass through the slot.

```html
<template>
  <NbButtonUnderlineFill
    :nb-id="'nb-button-underline-fill-one'"
    :display="'b'"
    textColor="rgb(0, 105, 255)"
    textHoverColor="#ffffff"
    buttonColor="rgb(0, 105, 255)"
    buttonHoverColor="rgb(20, 51, 214)"
    textAlign="center"
    :tabIndex="0"
    :hasTabIndexEnter="true"
    :hasTabIndexSpace="true"
    ariaLabel="BotÃ£o com preenchimento de underline"
    @clicked="logEvent()"
  >
    <template #content>
      Default Text
    </template>
  </NbButtonUnderlineFill>
</template>

<script setup>
defineOptions({
  name: 'CompUnderlineFill',
});

const logEvent = () => {
  console.log('clicked');
};
</script>
```
